# Java开源生鲜电商平台(B2B平台)-netcai

Java开源生鲜电商平台(B2B平台)

网菜网是一个基于Java的生鲜电商平台。

1.主要提供买家与卖家的服务的一个平台。

2.有买家APP,卖家APP,以及管理后端以及接口API等

3.个人可以免费学习与使用代码。

4.商业用户需要支付使用费。

相关的内容简介与技术学习可以查看我的博客：

http://www.cnblogs.com/jurendage
